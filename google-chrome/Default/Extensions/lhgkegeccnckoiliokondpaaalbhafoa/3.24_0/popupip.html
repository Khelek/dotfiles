Default tab:

This extension displays the geolocation, DNS, whois, routing, hosting, domain neighbors, DNSBL, BGP and ASN information of every IP address (IPv4 and IPv6). For domains this extension displays SEO information like Alexa ranking, Quantcast ranking, social media activity, whois and PageRank.

IPv4: shows the IP Address Information (IPv4) from the current website.

IPv6: shows the IP Address Information (IPv6) from the current website.

ISP: shows the ISP information from the current website.

Domain: shows information about the domain of the current website in Chrome.

My IP: shows information about your public IP address.

Options: sets the default tab to load.
